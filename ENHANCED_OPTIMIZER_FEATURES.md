# Portfolio Optimizer Enhanced - 기능 설명

## 🎯 새로운 통합 버전의 특징

### Lovable AI 스타일 + 기존 프로젝트 장점 결합

## ✨ 주요 기능

### 1. **간단한 입력 폼 (Lovable 스타일)**
- ✅ **Stock Tickers**: 직접 입력 또는 검색으로 추가
- ✅ **Investment Amount**: 투자 금액 입력 (환율 변환 지원)
- ✅ **Risk Tolerance**: 드롭다운 선택
  - Conservative (0.2)
  - Medium - Balanced (0.5) - 기본값
  - Aggressive (0.8)
- ✅ **Optimization Method**: 양자/클래식 선택
- ✅ **Data Period**: 1개월, 3개월, 6개월, 1년

### 2. **Dashboard 연동 (기존 기능)**
- ✅ Dashboard 포트폴리오 자동 불러오기
- ✅ 체크박스로 간편 전환
- ✅ 포트폴리오 가치 자동 설정

### 3. **주식 검색 기능 (기존 기능)**
- ✅ 실시간 주식 검색
- ✅ 한국/미국 주식 지원
- ✅ 검색 결과에서 클릭으로 추가
- ✅ 실시간 주가 위젯 표시

### 4. **환율 변환 (기존 기능)**
- ✅ 언어에 따라 KRW/USD 자동 변환
- ✅ Investment Amount에 환율 적용
- ✅ 실시간 환율 반영

### 5. **양자/클래식 비교 (기존 기능)**
- ✅ 개별 최적화 실행
- ✅ 동시 비교 실행
- ✅ 상세한 메트릭 비교
- ✅ 바 차트 시각화

### 6. **결과 시각화 (기존 기능 + 개선)**
- ✅ 파이 차트 (최적화된 비중)
- ✅ 주식별 배분 금액
- ✅ 예상 수익률, 위험도, 샤프 비율
- ✅ 비교 뷰 (양자 vs 클래식)

## 🎨 디자인 특징

### Lovable AI 스타일
- 다크 테마 (gray-900 → gray-800)
- 그린 악센트 컬러 (#00B894)
- 깔끔한 2컬럼 레이아웃
- 초기 상태 "Get Started" 메시지

### 기존 프로젝트 스타일
- 상세한 차트 및 시각화
- 백엔드 연결 상태 표시
- 다국어 지원
- 반응형 디자인

## 📊 사용 방법

1. **티커 입력**:
   - 직접 입력: "AAPL, GOOGL, MSFT"
   - 또는 검색으로 추가

2. **투자 금액 설정**:
   - 원하는 투자 금액 입력
   - 환율 자동 변환 표시

3. **위험 허용도 선택**:
   - 드롭다운에서 선택
   - 각 옵션 설명 표시

4. **최적화 실행**:
   - 단일 방법 실행
   - 또는 비교 실행

5. **결과 확인**:
   - 오른쪽 패널에서 결과 확인
   - 차트 및 통계 확인

## 🔄 기존 컴포넌트와의 차이

| 기능 | 기존 PortfolioOptimizer | Enhanced 버전 |
|------|------------------------|---------------|
| 입력 방식 | Dashboard에서만 불러오기 | 직접 입력 + Dashboard 연동 |
| Investment Amount | 없음 | 있음 (환율 변환) |
| Risk Tolerance | 슬라이더 (0.0-1.0) | 드롭다운 (3가지 옵션) |
| 초기 상태 | 포트폴리오 없음 메시지 | "Get Started" 안내 |
| 디자인 | 라이트 테마 | 다크 테마 + 그린 악센트 |
| 레이아웃 | 3컬럼 | 2컬럼 (입력/결과) |

## 🚀 접근 방법

- **새 버전**: Sidebar → Optimizer (기본)
- **기존 버전**: `optimizer-classic` 라우트 (유지)

## 💡 향후 개선 사항

1. 결과 내보내기 (PDF, Excel)
2. 포트폴리오 저장/불러오기
3. 최적화 히스토리
4. 실시간 포트폴리오 모니터링
