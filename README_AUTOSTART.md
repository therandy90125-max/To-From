# 🚀 QuantaFolio Navigator 자동 시작 설정

## 📋 개요

Windows 부팅 시 자동으로 QuantaFolio Navigator를 시작하도록 설정할 수 있습니다.

## 🔧 설치 방법

### 방법 1: 자동 설치 스크립트 (권장)

```batch
install-autostart.bat
```

이 스크립트를 실행하면:
- 시작 프로그램 폴더에 바로가기가 생성됩니다
- 다음 Windows 부팅 시 자동으로 시작됩니다

### 방법 2: 수동 설정

1. `Win + R` 키를 눌러 실행 창 열기
2. `shell:startup` 입력 후 Enter
3. `start-all.bat` 파일의 바로가기를 시작 프로그램 폴더에 복사
4. 바로가기 속성에서 "인수"에 `--auto` 추가 (선택사항)

## 🗑️ 제거 방법

### 방법 1: 자동 제거 스크립트

```batch
uninstall-autostart.bat
```

### 방법 2: 수동 제거

1. `Win + R` 키를 눌러 실행 창 열기
2. `shell:startup` 입력 후 Enter
3. "QuantaFolio Navigator" 바로가기 삭제

## ⚙️ 동작 방식

- **자동 시작 모드 (`--auto`)**: 
  - 백그라운드로 실행
  - 창을 닫으면 서비스가 종료됨
  - 서비스를 계속 실행하려면 창을 열어두어야 함

- **수동 실행 모드**:
  - 기존과 동일하게 동작
  - 모든 서비스가 별도 창에서 실행됨

## 📝 참고사항

- 자동 시작은 현재 사용자 계정에만 적용됩니다
- 모든 사용자에게 적용하려면 관리자 권한이 필요합니다
- 서비스를 종료하려면 `stop-all.bat`를 실행하세요

## 🔍 문제 해결

### 자동 시작이 작동하지 않는 경우

1. 시작 프로그램 폴더에 바로가기가 있는지 확인
2. 바로가기 경로가 올바른지 확인
3. Windows 방화벽이 차단하지 않는지 확인
4. 관리자 권한으로 `install-autostart.bat` 실행

### 서비스가 자동으로 종료되는 경우

- 자동 시작 창을 닫으면 모든 서비스가 종료됩니다
- 서비스를 계속 실행하려면 창을 열어두세요
- 또는 각 서비스를 Windows 서비스로 등록하는 것을 고려하세요
